# Portfolio Website

A personal portfolio website built with Next.js (App Router), React, TypeScript, Tailwind CSS, Shadcn/UI, Framer Motion, and Magic UI. This repository is a customized configuration of the original [dillionverma/portfolio](https://github.com/dillionverma/portfolio) template.

## 🛠️ Features

- One‑file configuration for quick setup
- Built using Next.js 14, React, TypeScript, Shadcn/UI, TailwindCSS, Framer Motion, Magic UI
- Blog section ready to publish posts via Markdown
- Fully responsive design for mobile, tablet, and desktop
- SEO‑friendly and optimized for performance
- Easy customization: update site metadata, links, and content in the config file

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. **Navigate into the project directory**
   ```bash
   cd portfolio
   ```
3. **Install dependencies**
   ```bash
   pnpm install
   ```
4. **Run the development server**
   ```bash
   pnpm dev
   ```
5. **Open your browser** and go to `http://localhost:3000`

---

## 🔧 Configuration

All site content (name, bio, links, projects, experience, skills) can be customized in the single configuration file located at:

```
src/config.ts
```

Update this file with your own details, then restart the development server to see changes.

---

## 📦 Deployment

This project uses Next.js App Router and requires a Node.js environment.

1. **Build for production**
   ```bash
   pnpm build
   ```
2. **Start the production server**
   ```bash
   pnpm start
   ```

You can deploy to any Node.js‑compatible host (Vercel, Hostinger Premium/VPS, DigitalOcean, etc.).
